# BOF (Bling_On_ofF)

Arduino library for playing with relay, LED, and other On/OFF.

## Description
BOF (Bling_On_ofF) is an small Arduino library for relay, LED, and other On/OFF thing. 

Void for the moment :
- on() to turn on the relay 
- off() to turn off
- bling(float,float)
- fade(float) 
- isEnable()
- isDisable()
- isPause()
- BPM()
and
- process()
